# FreeTabs Framework

基于管理系统的业务复杂性，多页签框架是当前的主流。然而，VueRouter的单实例性使得大部分管理系统无法做到同类型页面多开。该框架实现了一套多实例多页签管理系统的路由框架。页面路由基于单个标签页进行,支持标签页路由重定向、标签页回退、新标签页启动、带参启动、同类页面支持多个实例、演示模式等，可以调用外层框架进行交互的API。
